  TIPS:
  • Compress JPGs to under 400KB each for fast loading.
    Use https://squoosh.app or similar free tools.
  • Use PNG only for logos (needs transparency).
  • Keep all filenames lowercase — the website is case-sensitive.
  • If an image file is missing, a dashed placeholder label
    will appear in its place — the site still works fine.
  • Gallery images on Charcoal and Agriculture pages open
    in a full-screen lightbox when clicked.
